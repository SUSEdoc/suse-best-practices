Installing Java

Oracle WebLogic Server 12cR2 (12.2.1) requires JDK version 1.8. Log in to the target system (SUSE Linux Enterprise Server 12 64-bit operating system) as a non-administration user. Download the Java SE Development Kit 8 (jdk-8u91-linux- x64.tar.gz) from http://www.oracle.com/technetwork/indexes/downloads/index.html#java.

Set the environment variables JAVA_HOME and PATH to ensure the proper JDK version is installed and ready for use.
