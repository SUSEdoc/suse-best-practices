Setting Up SMT

The SMT service provides a local repository mirror for updates to your software.  Follow the instructions in the Subscription Management Tool Guide at https://documentation.suse.com/sles/12-SP4/html/SLES-all/smt-installation.html to install and configure SMT. Be sure to select both the pool and update repositories for each product you are supporting with this server.
